# LLMGEO - 海外 GEO 排名优化 SaaS 系统完整开发文档

## 项目概述

### 项目基础信息
- **项目名称**：LLMGEO
- **产品定位**：多租户 SaaS 平台，依托多厂商大模型 AI 自动生成品牌图文/视频内容，批量分发至全球社媒、海外站群，沉淀外链提升 Google 收录与海外 GEO 搜索排名；分销代理商付费体系，配套全链路跨境增值预约服务，私有化私有部署不开源。
- **UI风格参考**：semrush.com
- **大模型**：Agnes API (Base URL: https://apihub.agnes-ai.com/v1)
- **部署方案**：GitHub 私有仓库 + Cloudflare Workers/D1/R2/Queues（零云服务器）

### 技术栈约束（Cloudflare 原生方案）
- **运行环境**：Cloudflare Workers (Hono 框架)
- **数据库**：Cloudflare D1 (SQLite 兼容)
- **对象存储**：Cloudflare R2
- **消息队列**：Cloudflare Queues
- **定时任务**：Cloudflare Cron Triggers
- **CI/CD**：GitHub Actions
- **第三方集成**：多厂商 LLM API、OAuth2 社媒授权、支付接口、邮件通知
- **部署**：GitHub 私有仓库 + Cloudflare CDN/托管，全程不开源

## 四层用户角色权限体系

### 1. 总控管理员（平台拥有者，唯一超级权限）
- 全局数据看板：代理商总数、企业租户数量、AI 素材生成总量、外链发布总量可视化统计
- 全量财务管控：代理商充值/扣款流水、全部付费订单、AI 套餐账单、增值预约 6 元订单明细
- 数据导出权限：按日/按月/全量导出 AI 文章、图片、视频压缩包；批量导出全部发布外链 CSV（对接蜘蛛池）
- 平台全局配置：AI 套餐定价、社媒 App 密钥、各大模型公共 API、邮件通知模板、存储配置、支付参数
- 全账号管理：查看所有代理商、代理商名下企业、企业运营子账号；冻结、注销任意账户
- 工单管理：查看全部增值预约表单，新增工单自动邮件告警管理员
- 日志审计：全平台操作日志、AI 调用日志、社媒发布失败日志、财务流水日志

### 2. 代理商账户（开户付费 8888 元）
- 分销开户能力：后台自主创建企业租户，自动扣除代理商预存余额 888 元/企业/年
- 企业开通 7 天内可提交注销退款申请，管理员审核通过原路返还余额
- 客户数据管理：名下企业列表、企业开通时间、会员到期邮件提醒、单企业发布明细、总发布数据看板
- AI 工具权限：内置全系列大模型对话窗口，侧边栏关键词一键复制检索 GEO 排名效果
- 增值服务特权：8 类跨境增值预约表单免费提交，无需支付 6 元预约费
- 数据隔离：仅可查看自身代理下所有企业数据，无权限访问其他代理商、总控财务后台

### 3. 企业用户（租户，公司名称全局唯一）
- **两种开通渠道**：
  - 代理商后台代开：年费 888 元，扣代理商预存款
  - 官网前台自主注册：年费 1688 元
- **基础业务功能**：
  - 企业信息管理：唯一企业名称、官网、品牌名、联系方式（邮箱/电话/WhatsApp）增删改
  - 关键词管理：产品关键词无上限新增、编辑、批量删除
  - 社媒账号绑定：国内外全平台 OAuth 扫码授权，统一存储令牌用于自动分发内容
- **AI 内容自动生产规则**：
  - 开户赠送平台通用大模型免费使用 3 天
  - 3 天后必须配置企业自有大模型 API，无自有密钥则暂停 AI 生成任务
  - 付费购买平台通用模型包：66 元/天、666 元/月，欠费弹窗拦截 AI 功能
- **自动内容分发渠道**：
  - 自有绑定社媒账号
  - LLMGEO 官方站群：Wordpress 站群、Blogger、Google Sites、Medium、Tumblr、Substack、Bear Blog
- **数据看板首页**：发布总量图表、外链集合快速入口、多模型内置聊天窗、关键词侧边栏、AI 素材库（图片/视频本地下载）
- **子账户管理**：单企业最多创建 10 名运营人员子账号
- **增值预约**：8 类跨境配套服务，每单需扫码支付 6 元提交预约工单
- **数据权限**：仅查看本企业所有素材、发布外链、预约工单

### 4. 运营人员（企业子账号，上限 10 人/企业）
- **可操作权限**：编辑本企业基础资料、关键词、查看发布记录、预览下载 AI 素材、使用内置 AI 聊天窗口
- **禁止权限**：充值续费、新增企业、提交增值预约工单、查看财务数据、切换其他企业数据
- **强数据隔离**：仅可见所属企业数据，无跨企业访问权限

## 付费与财务完整流程

1. **代理商开通**：一次性支付 8888 元 → 激活代理权限，生成独立预存余额池
2. **企业代开扣费**：代理商后台新建企业租户，自动扣减余额 888 元/年
3. **企业自主注册**：前台支付 1688 元/年，直接创建租户账号
4. **退款规则**：企业开通 7 天内代理商提交注销申请，总控审核通过后返还对应扣费金额至代理商余额
5. **AI 模型付费套餐**：66 元单日包、666 元月包，企业单独付费，独立账单记录
6. **增值预约收费区分**：
   - 企业用户：8 类预约表单每单 6 元，支付后生成工单
   - 代理商后台提交预约：免费，无需支付任何费用

## 核心功能模块清单

### 模块 1：多租户账户与 RBAC 权限模块
- 用户注册、登录、密码重置、邮箱通知
- 四级角色路由守卫、细粒度权限控制
- 全局租户隔离中间件，拦截跨租户越权访问
- 代理商余额管理、充值、扣款事务逻辑
- 企业租户开通、续费、注销退款申请流程
- 运营子账号增删，单企业 10 人上限校验
- 会员到期定时检测、邮件提醒任务
- 全平台账号冻结、注销管控（仅总控）

### 模块 2：企业信息 & 关键词管理模块
- 企业基础资料 CRUD，企业名称全局唯一校验
- 产品关键词无限新增、批量编辑、批量删除
- 联系方式（邮箱/电话/WhatsApp）管理
- 数据按企业租户隔离存储

### 模块 3：统一 OAuth2 社媒授权模块（核心难点）
- 基于 league/oauth2-client 封装全平台统一授权适配器
- 强制离线刷新 token 授权 scope，永久存储令牌至数据库
- 定时后台任务自动刷新 access_token
- 令牌失效前端红色提醒，一键重新扫码授权
- 统一多平台内容发布适配接口，自动适配图文/视频上传参数限制
- 企业仅可解绑、查看自身绑定社媒账号
- **社媒同步登录 & 发布解决方案**：
  - 企业后台点击平台图标跳转第三方 OAuth 授权页面，企业扫码确认授权
  - 系统存储该企业独立永久刷新令牌，无需企业重复扫码
  - AI 生成内容完成后，调用各平台官方 API 自动推送图文/视频
  - 令牌过期自动标记失效，进入社媒管理页面弹窗提示重新授权

### 模块 4：LLM 多模型 AI 生成引擎（系统核心）
- 统一多模型适配器，兼容全部文本、多模态 API
- 企业自有 API 密钥录入、切换管理页面
- 平台通用 AI 套餐购买、有效期校验、欠费拦截中间件
- 任务队列批量生成 AI 内容，分离文本/图片/视频队列
- 生成内容自动植入企业品牌、产品关键词、联系方式
- AI 素材（图片/视频）异步上传 Cloudflare R2，存储路径入库
- AI 生成文章、媒体素材分表存储，严格按企业租户隔离
- 企业首页内置多模型聊天窗口，侧边栏关键词一键复制至对话输入框

### 模块 5：自动分发发布 & 外链收集导出模块
- 调度服务统一分发内容至自有社媒 + LLMGEO 站群渠道
- 发布成功自动抓取页面完整外链 URL 存入发布记录表
- 发布失败日志记录，后台支持手动批量重发失败任务
- 分级外链查看权限：企业看自身外链、代理商看名下企业外链、总控查看全平台外链
- 外链 CSV 批量导出功能；总控支持按日/月导出全部 AI 文章、图片、视频压缩包，供给蜘蛛池系统爬取收录

### 模块 6：AI 素材库管理模块
- 企业隔离素材展示，预览图片、播放视频
- 单文件本地下载、批量打包下载
- 总控后台全平台素材筛选打包导出

### 模块 7：首页可视化看板模块（企业/代理商）
- ECharts 发布数量统计图表
- 外链集合页面快捷入口
- AI 素材库快捷入口
- 会员到期、AI 套餐欠费弹窗提醒
- 内置多模型对话窗口、关键词复制侧边栏

### 模块 8：增值预约工单系统
- 统一表单渲染逻辑，区分企业付费 6 元、代理商免费提交两种流程
- 工单状态管理：待联系、处理中、已完成
- 新增工单自动发送邮件通知总控管理员
- 工单筛选、Excel 导出功能
- 权限隔离：企业仅查看自身工单，代理商查看名下企业全部工单，总控查看全量工单

### 模块 9：总控管理员全局后台模块
- 全平台数据总览看板
- 代理商列表穿透查看下级企业数据
- 全平台财务流水、订单筛选导出
- 系统全局参数配置页面（密钥、定价、存储、邮件、支付）
- 全量素材、外链批量打包下载
- 系统操作日志、AI 调用日志、发布日志审计
- 全平台账号冻结、注销管控面板

### 模块 10：Cloudflare 私有部署配套模块
- Cloudflare R2 对象存储文件上传、鉴权下载、防盗链
- GitHub Actions CI/CD 自动化部署脚本
- 环境变量密钥隔离，敏感配置不上传代码仓库
- 静态资源 CDN 加速、全球 HTTPS、海外访问防护规则

## D1 数据库核心表设计

### 1. sys_admin 总控管理员表
存储平台超级管理员账号，全局唯一

### 2. sys_agent 代理商账户表
字段：代理 ID、登录账号、密码、预存余额、开户时间、8888 元付费状态、联系邮箱、到期提醒开关

### 3. sys_company 企业租户表
字段：企业 ID、agent_id（关联代理商）、企业名称（唯一索引）、官网、品牌名、联系方式、会员年费到期时间、平台 AI 套餐有效期、租户唯一标识 tenant_id

### 4. sys_company_operator 运营子账号表
字段：operator_id、company_id、账号密码、姓名、创建时间；单 company_id 最多 10 条数据

### 5. sys_role 角色表
角色分类：admin 总控、agent 代理商、company 企业、operator 运营人员

### 6. sys_permission 细粒度权限码表
存储所有页面、接口权限标识，用于路由拦截

### 7. sys_user_role 用户角色关联表
绑定账号与对应角色权限

### 8. company_keyword 企业关键词表
tenant_id 隔离，存储产品关键词文本

### 9. company_social_oauth 社媒授权令牌表
tenant_id、平台标识、access_token、refresh_token、过期时间、授权用户 ID、绑定时间

### 10. ai_model_config 企业大模型配置表
tenant_id、模型类型、自有 API 密钥、平台套餐购买记录、套餐到期时间

### 11. ai_generate_content AI 生成文章主表
tenant_id、关键词 ID、生成文本内容、品牌信息、联系方式、生成时间、使用模型、关联媒体素材 ID

### 12. ai_media_file AI 多媒体素材表
tenant_id、文件类型（图片/视频）、R2 存储路径、文件大小、关联文章 ID、生成时间

### 13. publish_record 发布记录表
tenant_id、文章 ID、素材 ID、发布平台、完整外链 URL、发布时间、发布状态、失败原因

### 14. finance_order 全平台付费订单表
订单类型：代理开户、企业年费、AI 日/月套餐、增值预约 6 元单；支付金额、支付状态、创建时间、关联租户/代理商 ID

### 15. agent_balance_log 代理商余额流水表
充值记录、企业代开扣费记录、退款返还记录、余额变动前后数值、操作时间

### 16. reservation_form 增值预约工单表
租户/代理商标识、8 类服务类型、申请人、联系方式、需求描述、使用人数、期望联系时间、支付状态、工单处理状态、创建时间

### 17. system_config 系统全局配置表
平台定价、各大模型公共 API、社媒 AppID/AppSecret、邮件模板、R2 存储参数、支付接口配置

### 18. system_log 系统操作日志表
登录、扣款、AI 调用、账号变更、发布操作全量日志留存

## 部署约束规范

- GitHub 仓库强制私有，禁止开源、禁止 Fork、禁止公开代码
- 所有密钥、数据库账号、API 密钥统一存放 Cloudflare 环境变量 / GitHub Action Secrets，禁止硬编码
- AI 图片、视频素材存放 Cloudflare R2 私有桶，后端鉴权访问，禁止公开直链
- 全站强制 HTTPS，Cloudflare 开启海外 CDN 优化
- 定时任务、队列常驻后台运行
- 所有付费订单、财务流水、操作日志长期留存，支持导出对账审计
